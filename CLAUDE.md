# Instrucciones para Claude Code

## Convenciones del proyecto

Antes de sugerir nombres de ramas o mensajes de commit, **siempre consultar**:
- `docs/convencion-ramas.md` — convención de nombres de ramas (prefijos: feature/, bugfix/, hotfix/, release/, chore/, docs/, test/)

Los mensajes de commit siguen el estándar **Conventional Commits**:
`tipo(scope): descripción` donde tipo es: feat, fix, docs, chore, refactor, test.
